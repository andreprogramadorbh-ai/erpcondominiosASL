Como configurar o Gmail

  • Use smtp.gmail.com, porta 587, protocolo TLS.
  • No campo Senha do App Google, insira a Senha de App de 16 caracteres gerada em myaccount.google.com/apppasswords (sem espaços).
  • A senha é armazenada criptografada no banco de dados com AES-256.

Chave de Criptografia (APP_ENCRYPTION_KEY)

O que é e por que é necessária?

A APP_ENCRYPTION_KEY é uma chave secreta usada para criptografar a senha do servidor SMTP antes de armazená-la no banco de dados. Isso garante que, mesmo que o banco seja comprometido, a senha não seja exposta em texto puro.

  • Deve ser adicionada ao arquivo .env na raiz do servidor.
  • Nunca compartilhe ou exponha esta chave.
  • Se a chave for alterada, será necessário recadastrar a senha SMTP.
